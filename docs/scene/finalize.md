# Scene - Finalize

## Description

Finalizes the code and adds the scene code to the main code of the VN.

## Syntax

```js
scene.finalize(scene)
```

## Parameters

- `scene`: The scene configurations from the `init` function.

## Return value

This function will always return `undefined`.
