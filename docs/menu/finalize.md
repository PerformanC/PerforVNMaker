# Menu - finalize

## Description

Generated the menu code of the VN, **necessary** for you to be able to use the VN.

## Syntax

```js
menu.finalize(menu)
```

## Parameters

- `menu`: The menu object generated by `menu.init`.

## Return value

This function will always return `undefined`.
